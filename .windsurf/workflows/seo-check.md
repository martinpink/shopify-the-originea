# SEO Check — The Originea

## Check the following for $ARGUMENTS:

### 1. Title Tag
- [ ] Length: 50-60 characters
- [ ] Contains primary keyword
- [ ] Ends with "| The Originea" (not duplicated)
- [ ] In French

### 2. Meta Description
- [ ] Length: 140-160 characters
- [ ] Contains primary + secondary keywords
- [ ] Includes a call to action
- [ ] Mentions Provence or artisanal
- [ ] In French

### 3. H1
- [ ] Exactly one H1 per page
- [ ] Matches or closely mirrors the title tag keyword
- [ ] In French

### 4. Product Description
- [ ] Mentions key search terms naturally
- [ ] Includes variants/sizes
- [ ] Mentions biodégradable/naturel where applicable
- [ ] Has proper HTML structure (not just plain text)

### 5. Images
- [ ] All images have alt text
- [ ] Alt text is descriptive and in French
- [ ] No alt text duplicated across images

### 6. URL/Handle
- [ ] URL is clean and contains the primary keyword
- [ ] No stop words (le, la, les, de, etc.) if possible
- [ ] Uses hyphens not underscores

## Output
Report each check as ✅ PASS or ❌ FAIL with a brief fix suggestion for any failures.
