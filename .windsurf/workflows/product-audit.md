# Product Audit — The Originea

## Audit $ARGUMENTS for the following:

### Images
- [ ] At least 3 product images
- [ ] All images have alt text (not empty)
- [ ] Alt text is in French and descriptive
- [ ] No image filename is generic (img_001.jpg etc.)

### Variants & SKUs
- [ ] Every variant has a SKU
- [ ] SKU follows The Originea format: `[CAT]-[CODE]-[VARIANT]`
- [ ] No duplicate SKUs
- [ ] Inventory tracking is enabled

### SEO
- [ ] Page title set (not using default product title)
- [ ] Meta description filled (140-160 chars)
- [ ] Meta description in French
- [ ] URL handle is clean and keyword-rich

### Description
- [ ] Description is not empty
- [ ] Uses proper HTML structure (not plain text)
- [ ] Mentions Provence or artisanal
- [ ] Includes shipping info (livraison offerte dès 45€)
- [ ] At least 150 words

### Collections
- [ ] Product belongs to at least one collection
- [ ] Product is not orphaned (accessible via navigation)

### Tags
- [ ] Has relevant tags for filtering
- [ ] Has `personnalisable` tag if applicable

## Output
Report each check as ✅ PASS or ❌ FAIL.
Give a priority score: 🔴 Critical / 🟡 Important / 🟢 Nice to have.
End with a summary: "X issues found — Y critical, Z important."
