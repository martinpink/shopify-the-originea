# Translate to French — The Originea Brand Voice

## Brand Voice Guidelines
- **Tone:** Warm, poetic, artisanal, premium but approachable
- **Style:** Evocative of Provence — nature, sun, authenticity
- **Avoid:** Corporate language, overly formal register, English loanwords where French exists
- **Use:** "Artisanal", "naturel", "Provence", "savoir-faire", "authenticité"

## Translation Rules
1. Translate $ARGUMENTS (or the provided text/file) to French
2. Adapt idioms naturally — don't translate literally
3. Keep any Liquid/HTML tags exactly as-is
4. Preserve emoji if present
5. Keep discount codes, URLs, and technical strings in their original form
6. For product descriptions: use sensory language (goût, parfum, texture)
7. For emails: use "vous" (formal) not "tu"
8. For UI buttons: short, action-oriented (e.g. "Commander", "Découvrir", "Ajouter au panier")

## Output
Provide:
1. French translation
2. Brief note if any adaptation was made (e.g. "idiom adapted", "tone adjusted")
